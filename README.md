# RFID Security System 🔐

## 📌 Overview
This project is an Arduino-based RFID security system used to control access.

## ⚙️ Components Used
- Arduino Uno
- RFID RC522 Module
- Servo Motor
- Buzzer
- IR Sensor

## 🚀 Working
1. RFID card is scanned
2. If authorized → Servo motor opens
3. If unauthorized → Buzzer alert
